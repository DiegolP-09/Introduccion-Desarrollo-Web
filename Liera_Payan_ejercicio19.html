Salidas con JavaScript

// Datos personales mostrados con los 4 métodos de salida

window.alert()

Cuadro de alerta

Muestra un cuadro de diálogo modal en el navegador.
Detiene la ejecución hasta que el usuario lo cierra.

window.alert("Nombre: Diego Liera Payan\nCarrera: Ing. en Software\nUniversidad: UAS\nSemestre: 6to");
document.write()

Escribir en el documento

Inserta HTML directamente en el flujo del documento.
Si se llama después de la carga, reemplaza el contenido.

document.write("<h3>Diego Liera Payan</h3>" + "<p>Matrícula: 2002599</p>" + "<p>Correo: estudianteuas@gmail.com</p>");
// La salida se mostrará aquí (simulada)

⚠ document.write() está obsoleto. Úsalo solo en ejercicios; en producción usa innerHTML.
console.log()

Consola del navegador

Imprime mensajes en la consola de desarrollo (F12).
Ideal para depuración; no visible para el usuario final.

const datos = { nombre: "Diego Liera Payan", carrera: "Ing. en Software", matricula: "2002599", semestre: "6to" }; console.log("=== Mis datos ==="); console.log(datos);
// Abre DevTools (F12) → pestaña Console

element.innerHTML

Modificar el DOM con innerHTML

Método recomendado para actualizar el contenido de un elemento HTML en tiempo real sin recargar la página.

// Seleccionamos el elemento destino const contenedor = document.getElementById("perfil"); // Inyectamos HTML con los datos personales contenedor.innerHTML = ` <strong>Nombre:</strong> Diego Liera Payan <br> <strong>Carrera:</strong> Ing. en Software <br> <strong>Matrícula:</strong> 2002599 <br> <strong>Semestre:</strong> 6to <br> <strong>Correo:</strong> estudianteuas@gmail.com <br> <strong>GitHub:</strong> github.com/DiegoIP-09 `;
// Haz clic en "Mostrar perfil" para ver la salida
