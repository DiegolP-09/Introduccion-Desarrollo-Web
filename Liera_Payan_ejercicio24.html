Fechas con JavaScript

Mostrar la fecha actual en texto usando la sentencia switch

Fecha de hoy
Calculando...

¿Cómo funciona?

1

new Date() crea un objeto con la fecha y hora actual del sistema.

2

getDay() retorna un número del 0 (domingo) al 6 (sábado). Con switch lo convertimos al nombre del día.

3

getMonth() retorna del 0 (enero) al 11 (diciembre). Otro switch lo convierte al nombre del mes.

4

getDate() da el día del mes (1–31) y getFullYear() el año completo.

5

Se arma el texto con un template literal y se muestra con innerHTML.

// 1. Fecha actual
const hoy = new Date();

// 2. Extraer valores
const numeroDia = hoy.getDay();      // 0-6
const numeroMes = hoy.getMonth();    // 0-11
const diaMes    = hoy.getDate();     // 1-31
const anio      = hoy.getFullYear(); // ej. 2026

// 3. Nombre del día
let nombreDia;
switch (numeroDia) {
  case 0: nombreDia = "domingo";    break;
  case 1: nombreDia = "lunes";      break;
  case 2: nombreDia = "martes";     break;
  case 3: nombreDia = "miércoles";  break;
  case 4: nombreDia = "jueves";     break;
  case 5: nombreDia = "viernes";    break;
  case 6: nombreDia = "sábado";     break;
}

// 4. Nombre del mes
let nombreMes;
switch (numeroMes) {
  case  0: nombreMes = "enero";       break;
  case  1: nombreMes = "febrero";     break;
  case  2: nombreMes = "marzo";       break;
  case  3: nombreMes = "abril";       break;
  case  4: nombreMes = "mayo";        break;
  case  5: nombreMes = "junio";       break;
  case  6: nombreMes = "julio";       break;
  case  7: nombreMes = "agosto";      break;
  case  8: nombreMes = "septiembre";  break;
  case  9: nombreMes = "octubre";     break;
  case 10: nombreMes = "noviembre";   break;
  case 11: nombreMes = "diciembre";   break;
}

// 5. Armar y mostrar el texto
const texto = `Hoy es ${nombreDia} ${diaMes} de ${nombreMes} del año ${anio}`;
document.getElementById("fecha-resultado").innerHTML = texto;
